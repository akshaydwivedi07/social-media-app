# 🚀 Social Media App

A modern full-stack social media application built using the MERN Stack with real-time communication, responsive UI, and interactive social features.

---

# ✨ Features

## 🔐 Authentication
- User Signup & Login
- JWT Authentication
- Persistent Login System
- Secure Logout

---

## 👤 User Profiles
- Upload Profile Picture
- Edit Bio
- Personalized User Profiles
- Followers / Following System

---

## 📸 Posts
- Create & Upload Posts
- Like / Unlike Posts
- Comment on Posts
- Delete Posts
- Dynamic Feed System

---

## 🔍 Search Users
- Search Users by Username
- Visit User Profiles
- Follow / Unfollow Users

---

## 💬 Real-Time Chat
- Instant Messaging using Socket.io
- Real-Time Updates
- Modern Chat Interface

---

## 📱 Responsive UI
- Mobile Responsive Design
- Smooth Animations
- Sidebar Navigation
- Modern Social Media Interface

---

# 🛠️ Tech Stack

## Frontend
- React.js
- CSS3
- React Icons
- Socket.io Client

## Backend
- Node.js
- Express.js
- Socket.io

## Database
- MongoDB Atlas
- Mongoose

## Authentication
- JWT (JSON Web Token)
